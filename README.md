# ExternHook
- A simple API, simular to MinHook, except its used for DLL injection.
- Currently, this is in beta. It is pretty unstable, as sometimes the process crashes.
- That might be my fault though, as I hooked on QueryPerformanceCounter, perhaps there was a better function.
- Im not sure, but what I do know, is that this works.
- Example program is in source/main.cpp. You can see how it works there.
